DISEASE-SPECIFIC RULES
================================================================================

DIABETES:
  • Max sugar: 5g per meal
  • Prefer: low-GI, high-fiber, low-sugar
  • Avoid: refined-carbs, sugary drinks

HYPERTENSION:
  • Max sodium: 400mg per meal
  • Prefer: low-sodium, potassium-rich, heart-healthy
  • Avoid: processed, canned foods

OBESITY:
  • Single meal max: 30% of daily calorie target
  • Prefer: low-calorie, high-protein, high-fiber
  • Avoid: fried, high-calorie, fast-food

PCOS:
  • Max sugar: 8g per meal
  • Prefer: anti-inflammatory, low-GI, high-fiber
  • Avoid: refined-carbs, processed

KIDNEY DISEASE:
  • Max protein: 15g per meal
  • Max sodium: 350mg per meal
  • Max potassium: 200mg, phosphorus: 150mg
  • Prefer: kidney-friendly, low-protein, low-sodium

RECOMMENDATION ALGORITHM
================================================================================

Phase 1: FILTERING
  Input: Disease, goal, dietary preference
  For each meal:
    • Check hard numeric limits (sodium, protein, sugar, etc.)
    • Filter by dietary preference tags (vegan must have "vegan" tag)
    • Discard meals violating restrictions
  Output: Filtered meal list

Phase 2: SCORING
  For each remaining meal:
    • Calorie score: -5% per kcal deviation from target
    • Macro score: -10% protein, -5% carbs, -7% fats deviation
    • Tag bonus: +10 per preferred tag, +15 for diet match
    • Total score = 100 + bonuses - penalties
  Output: Scored meals

Phase 3: RANKING
  Sort meals by score (descending)
  Return top N (typically 4-5)

Complexity: O(n) where n = meal count (~100-500 meals)
Response time: <100ms for typical recommendations

RUNNING LOCALLY
================================================================================

1. Navigate to project:
   cd meal_recommendation_system

2. Create virtual environment:
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Setup MongoDB Atlas:
   - Create account at mongodb.com/cloud/atlas
   - Create M0 cluster (free)
   - Whitelist your IP (0.0.0.0 for local testing)
   - Create database user
   - Copy connection string

5. Configure environment:
   cp .env.example .env
   # Edit .env with MongoDB URI

6. Run app:
   streamlit run app.py

7. Open browser:
   http://localhost:8501

DEPLOYMENT ON STREAMLIT CLOUD
================================================================================

1. Push to GitHub (with meal_recommendation_system/ folder structure)

2. Go to share.streamlit.io

3. Click "New app" → Select repo & main file:
   meal_recommendation_system/app.py

4. In app settings → Secrets, add:
   MONGODB_URI = "mongodb+srv://..."
   MONGODB_DB = "meal_recommendation_db"

5. Deploy!

Live URL will be: https://username-repo-slug.streamlit.app
